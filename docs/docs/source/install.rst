Installation
============

Install via PyPI:

.. code-block:: bash
  :linenos:
  
    pip install spatialmeta

Install from source:

.. code-block:: bash
  :linenos:

    git clone git@github.com:WanluLiuLab/SpatialMETA.git
    cd spatialmeta
    pip3 install -r requirements.txt
    python3 setup.py install
